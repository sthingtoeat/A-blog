# 语法高亮

vuepress已经具备了Prism,你可以这样来使用 vuepress中使用Prism (opens new window)

引用代码块的时候 在```后面加上对应的语言类型即可

输入

```js
export default {
  name: 'MyComponent',
  // ...
}
```

输出

export default {
  name: 'MyComponent',
  // ...
}

# 提示标签

输入

::: tip
这是一个提示
:::

::: warning
这是一个警告
:::

::: danger
这是一个危险警告
:::

::: danger STOP
危险区域，禁止通行
:::

输出

TIP

这是一个提示

WARNING

这是一个警告

DANGER

这是一个危险警告

STOP

危险区域，禁止通行

# 收缩/展开

像这样输即可

::: details
这是一个 details 标签
:::

::: details 点击查看代码
```js
console.log('你好，VuePress！')
```
:::

效果：

DETAILS

这是一个 details 标签

点击查看代码 
console.log('你好，VuePress！')
